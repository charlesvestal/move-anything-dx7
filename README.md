# DX7 Module for Move Anything

Yamaha DX7 FM synthesizer emulator for Ableton Move, using the MSFA engine from Dexed.

## Features

- Full DX7 voice architecture
- Loads standard .syx patch banks
- 32 presets per bank
- Velocity and aftertouch support
- Signal Chain compatible

## Building

```bash
./scripts/build.sh
```

## Installation

Via Module Store (recommended):
- Launch Move Anything → Module Store → Sound Generators → DX7

Manual installation:
```bash
./scripts/install.sh
```
